MODULE 1

Basic Concepts of C#

C# Overview

C# is an object-oriented language designed for building applications on the .NET Framework.

.NET Framework Components

  • Common Language Runtime (CLR): Manages memory, code execution, and security.
  • Class Library: Predefined classes for tasks like file I/O and data processing.

Variables & Data Types

  • Primitive Types: int, float, double, char, bool, string
  • var keyword: Automatically infers variable type
  • Constants: Use const to declare unchangeable values

Input/Output

Console.WriteLine() – Outputs text with a line break

Console.ReadLine() – Reads user input (returns a string)

Type Conversion: Convert.ToInt32(), Convert.ToDouble()

Operators

Arithmetic: +, -, *, /, %

Assignment: =, +=, -=

Increment/Decrement: ++, --

Precedence: Use () to control order

Control Structures

Conditionals

if / else: Executes code based on Boolean condition

switch: Tests variable against multiple values using case, break, default

Relational Operators: ==, !=, >, <, >=, <=

Loops

  • while, for, do-while
  • break: Exit loop
  • continue: Skip to next iteration

Logical Operators

&& (AND), || (OR), ! (NOT)

Methods

Definition

Reusable blocks of code with a return type, name, and optional parameters

Advantages: Reusability, readability, debugging

Parameters

  • Pass-by-Value: Copies value
  • Pass-by-Reference: Use ref keyword
  • Output Parameter: Use out to return values

Overloading & Recursion

Overloading: Same method name with different parameters

Recursion: Method calls itself (e.g., factorial)

Object-Oriented Programming (OOP)

Classes & Objects

Class: Blueprint (fields + methods)

Object: Instance of a class

Encapsulation

Restrict access using public, private, protected

Properties: Controlled access with get and set

Constructors / Destructors

Constructor: Initializes an object

Destructor: Cleans up before object is destroyed

Static Members

Belong to the class, not instances (e.g., static int count;)

Advanced OOP

Inheritance

Base/derived class relationship using :

protected: Accessible in derived class

sealed: Prevents inheritance

Polymorphism

virtual: Declares base method

override: Overrides base method

Abstract Class

Cannot be instantiated; may include abstract methods (no body)

Interface

Fully abstract; a contract that must be implemented

Data Structures

Arrays

  • int[] nums = new int[5];
  • int[,] matrix = new int[3,3];
  • Methods: Sort(), Reverse(), Length

Strings

Useful methods: Length, IndexOf(), Substring(), Replace()

Advanced Concepts

Structs

Lightweight value types, no default constructor, passed by value

Enums

Set of named constants: enum Days { Monday, Tuesday }

Exception Handling

Use try-catch-finally blocks

Common exceptions: FileNotFoundException, IndexOutOfRangeException

File I/O

  • File.WriteAllText(path, content)
  • File.ReadAllText(path)
  • File.Delete(path)

Key Takeaways

  • Master the Basics: Data types, I/O, operators, control flow
  • Understand OOP: Encapsulation, inheritance, polymorphism
  • Work with Data: Arrays, strings, collections
  • Handle Errors: Exception handling and file operations

MODULE 2

Introduction to JavaScript

What is JavaScript?

A powerful scripting language used to create dynamic and interactive web content. Also used for mobile/desktop apps and games.

Adding JavaScript to Web Pages

  • Embedded using <script> tags (in <head> or <body>)
  • External files (.js) improve organization and performance

Comments

  • Single-line: // comment
  • Multi-line: /* comment */

JavaScript Basics

Variables

Declared with var, let, and const. Must start with a letter, _, or $.

Data Types

  • Number: Integers, decimals
  • String: Text like 'hello' or "world"
  • Boolean: true or false

Operators

  • Arithmetic: +, -, *, /, %, ++, --
  • Assignment: =, +=, -=
  • Comparison: ==, ===, !=, <, >, <=, >=
  • Logical: &&, ||, !
  • Ternary: condition ? value1 : value2

Conditionals and Loops

Conditionals

  • if, if-else, if-else-if
  • switch with case, break, and default

Loops

  • for: initializer, condition, increment
  • while: repeats while condition is true
  • do-while: runs once, then checks condition

Control Statements

  • break: exits loop or switch
  • continue: skips to next iteration

Functions

Definition

Reusable block of code: function name() { }

Parameters & Arguments

Values passed into functions to customize behavior.

Return Values

Use return to send output from a function.

Popup Boxes

  • alert("message") – simple notification
  • prompt("Enter name") – get user input
  • confirm("Are you sure?") – OK/Cancel dialog

Objects

Definition

Key-value pairs: { name: "John", age: 30 }

Methods

Functions inside objects

Built-in Objects

  • Math: Math.random(), Math.PI
  • Date: new Date(), getFullYear()

Arrays

Definition

Ordered list: let arr = [1, 2, 3]

Access

By index (starts at 0): arr[0]

Common Methods

  • length – number of elements
  • concat() – merge arrays

DOM (Document Object Model)

Overview

HTML is represented as a tree of objects

Selecting Elements

  • getElementById("id")
  • getElementsByClassName("class")
  • getElementsByTagName("tag")

Modifying Elements

  • element.innerHTML = "New content"
  • element.style.color = "red"

Event Handling

Attach events: onclick, onmouseover, onload

jQuery

Purpose

JavaScript library that simplifies DOM manipulation and event handling

Syntax

Starts with $. Example: $("p") selects all paragraph tags

Selectors

  • $("#id") – select by ID
  • $(".class") – select by class
  • $("div > p") – hierarchical selection

Key Takeaways

  • JavaScript enables interactive, responsive web pages
  • jQuery simplifies DOM tasks
  • Master variables, functions, objects, and the DOM for success in web dev
