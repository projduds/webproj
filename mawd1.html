Java Programming Notes

Basic Concepts

Introduction to Java: Java is a platform-independent, portable, robust, and dynamic language used for various applications.

Comments in Java

Variables

Types include: int, double, String, char, boolean.

Operators

Arithmetic: +, -, *, /, %

Increment/Decrement: ++, --

Assignment: =, +=, -=, etc.

Strings

Sequences of characters; concatenated using +.

User Input

Use Scanner class:

nextInt(), nextLine(), nextDouble(), etc.

Conditionals and Loops

Conditional Statements

if, if-else, else if, nested if

Logical operators: && (AND), || (OR), ! (NOT)

Switch Statement

Used to compare a variable against multiple values with case, break, and default.

Loops

  • while: Repeats as long as condition is true
  • do-while: Executes once before checking condition
  • for: Includes initialization, condition, and update
  • Loop control: break (exit loop), continue (skip iteration)

Arrays

One-Dimensional Arrays

int[] arr = new int[5];
int[] arr = {1, 2, 3};
System.out.println(arr[0]);
System.out.println(arr.length);

Enhanced For Loop

for (int x : arr) {
  System.out.println(x);
}

Multidimensional Arrays

int[][] matrix = {
  {1, 2},
  {3, 4}
};

Java Strings

Objects that store sequences of characters. Concatenation with +.

Special Characters

Escaped with backslash \:

\n = newline
\t = tab, etc.

Printing Output

System.out.print() – no newline

System.out.println() – with newline

MODULE 2

HTML & CSS Fundamentals

1. HTML Document Structure

<!DOCTYPE html>: Declares the HTML version (HTML5).

<html>: Root element for an HTML document.

<head>: Contains metadata like title, character encoding, and linked resources.

<title>: Sets the title of the webpage (appears on browser tab).

<meta charset="UTF-8">: Specifies character encoding (supports special characters).

<body>: Contains the content that displays in the browser.

2. HTML5 Semantic Elements

<header>: Introductory content or site branding.

<nav>: Navigation links.

<article>: Self-contained content (e.g., blog post).

<section>: Groups related content.

<aside>: Sidebar content related to the main content.

<footer>: Footer information (e.g., contact, copyright).

3. Text Formatting and Elements

<h1> to <h6>: Headings.

<p>: Paragraph.

<br>: Line break.

<hr>: Horizontal line.

<b> / <strong>: Bold text.

<i> / <em>: Italicized text.

<u>: Underlined text.

<s> / <del>: Strikethrough/deleted text.

<sup> / <sub>: Superscript / subscript.

4. Lists

<ol>: Ordered list.

<ul>: Unordered list.

<li>: List item.

<dl>: Definition list.

<dt>: Term in definition list.

<dd>: Definition of the term.

5. Tables

<table>, <caption>, <thead>, <tbody>, <tfoot>

<tr>: Table row.

<th>: Header cell.

<td>: Table cell.

colspan, rowspan: Merge cells.

border, cellpadding, cellspacing: Table appearance control.

6. Forms and Inputs

<form>, <fieldset>, <legend>

<label>, <input>, <select>, <option>, <textarea>

<button>, <input type="submit">, <input type="reset">

HTML5 Input Types: email, date, tel, color, range

Common Attributes: required, placeholder, pattern, min, max, autofocus

7. Multimedia

<img>: Images

<audio>: Audio

<video>: Video

<source>: Media source

alt: Alternative text

CSS Fundamentals

1. Syntax and Selectors

Selector: Element/class/id

Property: CSS style rule

Value: What is assigned (e.g., blue, 10px)

2. CSS Box Model

Content, Padding, Border, Margin

box-sizing: content-box (default), border-box

3. Text and Font Styling

font-family, font-size, font-weight, font-style

text-align, text-decoration, text-transform

letter-spacing, word-spacing, text-shadow

color: Text color

4. Backgrounds

background-color, background-image, background-repeat

background-position, background-size, background-attachment

background-origin, background-clip

5. Layout and Positioning

display: block, inline, flex, grid

position: static, relative, absolute, fixed, sticky

top, left, right, bottom

z-index, float, clear

CSS3 Advanced Features

1. Transitions

transition-property, transition-duration, transition-timing-function, transition-delay

2. Transforms

transform, transform-origin, transform-style

perspective, backface-visibility

3. Animations

@keyframes, animation-name, animation-duration

animation-delay, animation-timing-function, animation-iteration-count

animation-direction, animation-fill-mode, animation-play-state

4. Advanced Selectors

[attribute], [attribute=value], [attribute^=value], [attribute$=value], [attribute*=value]

:nth-child(n), :first-child, :empty

5. Media Queries

@media screen and (max-width: 600px)

@media print, @media (orientation: portrait), @media (min-resolution: 300dpi)

CSS Variables

--variable-name: Declares a custom variable.

var(--variable-name): Uses a CSS variable.

:root: Global scope for variables.
